A single-page e-commerce website built from scratch for soyjaking, an independent art and merchandise brand with 124,000+ Instagram followers, serving as the official storefront for a limited-run plushie designed, produced, and shipped by the creator. The site features a custom classic imageboard-inspired aesthetic with monospace fonts, amber headers, post-style sections, responsive design, mobile swipe gallery support, anchor navigation, and Stripe-powered secure checkout with shipping collection. Built using HTML, CSS, and JavaScript, with Netlify hosting, VScode, GitHub version control, and AI-assisted development through Claude. This project was my first coding experience and taught me the fundamentals of web development, project structure, payment integration, version control, DNS, deployment, and using AI as a collaborative tool throughout the build process.
